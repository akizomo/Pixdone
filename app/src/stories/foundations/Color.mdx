# Color Tokens

Pixdone uses a modern color system designed for readability while allowing playful retro accents. Use semantic tokens in components; reserve brand colors (epic, freeze, smash) for rewards and special moments.

## CSS variables

Use the `--pxd-` prefix in CSS: e.g. `--pxd-color-surface-page`, `--pxd-color-text-primary`. See `tokens.css`.
