# Spacing Tokens

Pixdone uses a consistent spacing scale for padding, margin, and layout gaps. Use **spacing** for component-internal spacing and **layoutSpacing** for page/section rhythm. Pair with `--pxd-tap-target-min: 44px` for minimum tap targets.
